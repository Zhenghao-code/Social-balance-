# ⚖️ 社交天平 (Social Balance) 

> **“在感情的‘四轮定位’中，数据比直觉更诚实。”**


还在为感情中付出太多而迷茫吗？**Social Balance** 是一款专为追求平等关系的人打造的纯本地聊天记录复盘工具。它通过解析微信导出的 JSON 数据，量化双向的话题发起频率，将隐形的情感投入转化为直观的物理天平。

帮你看清：**这段关系里，到底是谁在努力维系“扭矩”？**

---

## ✨ 核心功能 (Core Features)

- **📈 双向话题审计**：基于“4小时静默期”算法，精准判定每一次对话的“第一推力”归属。
- **🔍 开场白溯源**：一键查看我/对方发起话题时的前 50 字内容，识别是真诚分享还是工具性索取。
- **🚮 数据手术刀**：支持手动剔除“拍一拍”、系统提醒、敷衍表情包等干扰项，数据实时动态校准。
- **🛡️ 极致隐私防护**：采用纯前端 React 解析方案，数据仅在浏览器本地内存运行，**永不上传服务器**，安全无痕。
- **⚡ 高并发支持**：深度优化底层逻辑，轻松处理 10,000+ 条以上的海量聊天记录。

## 🚀 快速开始 (Quick Start)

### 1. 获取数据
目前本项目完美适配 **CipherTalk** 或类似微信解密工具导出的 `.json` 格式。
> *注：由于微信官方限制，获取 JSON 是最难的一步，但这正是量化真相的开始。*

### 2. 在线运行
1. 下载本项目中的 `index.html`。
2. 在浏览器中直接双击打开（支持 Chrome/Edge/Safari）。
3. 拖入你的 `.json` 文件，即可瞬间生成分析仪表盘。
4. 点击“主动发起”可以查看具体的聊天内容

### 3.测试文件

### 💡 没有数据？先试试 Demo！

如果你暂时没有导出的微信 `.json` 文件，可以按以下步骤先行体验分析效果：

1. **下载示例文件**：前往仓库中的 [`example/demo.json`](./example/demo.json) 文件夹下载测试数据。
2. **一键体验**：将该文件直接拖入 [在线分析工具](你的GitHub-Pages链接) 的虚线框内。
3. **观察深度分析**：
   - **天平倾斜度**：直观查看话题发起的比例分布。
   - **对话质量**：观察对方主动发起话题时，是分享生活还是“工具性”诉求（如：修图、求助）。

> **提示**：该 Demo 数据模拟了一个典型的不对等社交场景，帮助你快速理解工具的分析维度。

### 4.数据准备
本工具目前深度适配由开源项目 [**CipherTalk**](https://github.com/ILoveBingLu/CipherTalk) 导出的微信聊天记录。

 如何获取 JSON 文件？
1. 使用 **CipherTalk** 解密并导出你与特定联系人的聊天记录。
2. 在导出选项中选择 `.json` 格式。
3. 将生成的 JSON 文件拖入本工具即可开始分析。
